# Background Music 🎵

## Setup Instructions

Place your background music file in this folder with the name:

```
background.mp3
```

## Requirements

- **Format:** MP3 (recommended for browser compatibility)
- **Duration:** 2-5 minutes (loops automatically)
- **Volume:** Will be automatically set to 30% (pleasant level)
- **Mood:** Soft, emotional, astronomy/space themed

## Recommended Music Style

- Ambient/atmospheric
- Instrumental (no vocals)
- Slow tempo
- Calming and emotional
- Space/universe themed

## Example Music Sources

You can find royalty-free music from:
- YouTube Audio Library
- Free Music Archive
- Incompetech
- Bensound
- Purple Planet Music

## Notes

- The music button will appear even if the file is missing
- No errors will be thrown if the file doesn't exist
- Music does NOT autoplay (user must click the button)
- Music continues throughout the entire website journey
- Button is mobile-compatible and responsive
